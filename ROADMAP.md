# Roadmap

- [ ] Admin Console 上新建的 Custom Metadata 的 name 必须以 `cs_` 开头，防止和系统 Metadata 冲突
- [ ] Custom Metadata 除了 name, type 等关键属性不允许修改，其他属性允许修改
- [ ] System Metadata 只有 Label 允许修改，其他属性不允许修改
- [ ] 新增 Record Type 和 State 概念
- [ ] 实现完整的 Lookup Field 功能
- [ ] 实现完整的 Metadata Field 功能
- [ ] 实现 Data Layer
- [ ] 实现 Page Layout / Page List 的配置能力
